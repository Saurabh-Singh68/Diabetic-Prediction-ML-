# Diabetes Prediction Dashboard

This project is a Machine Learning based Diabetes Prediction System developed using Python, Scikit-learn, and Streamlit. The model predicts whether a person is diabetic based on medical parameters such as glucose level, blood pressure, BMI, insulin level, and age.

## Features

* Interactive Streamlit Dashboard
* Real-time Diabetes Prediction
* Machine Learning Classification Model
* User-friendly Interface

## Technologies Used

* Python
* NumPy
* Pandas
* Scikit-learn
* Streamlit

## Dataset

Pima Indians Diabetes Dataset
